# ระบบสั่งอาหารผ่าน QR Code และจัดการหน้าร้าน (QR Code Menu & Ordering System)

ระบบสั่งอาหารผ่านการสแกน QR Code สำหรับลูกค้า พร้อมแผงควบคุมหลักสำหรับแอดมิน/เจ้าของร้าน ที่มาพร้อมกับการชำระเงินผ่าน PromptPay QR Code, ระบบเช็คสต็อกสินค้า, สรุปยอดรายวัน และการบันทึกข้อมูลกับ Google Sheets โดยอัตโนมัติ

---

## ฟีเจอร์เด่น
1. **หน้าสั่งอาหารสำหรับลูกค้า (Customer Portal)**
   - สแกน QR Code เพื่อเข้าสู่เมนูระบุโต๊ะอาหารอัตโนมัติ (เช่น โต๊ะ 1, 2, 3...)
   - ค้นหาเมนูอาหารและเลือกจำนวนใส่ตะกร้า (ป้องกันการสั่งเกินจำนวนสต็อกที่มีอยู่)
   - ส่งออเดอร์เข้าครัวได้แบบเรียลไทม์ผ่านมือถือ
   - ติดตามสถานะออเดอร์ของตนเองได้แบบไลฟ์ (รอยืนยัน, จ่ายเงินแล้ว, เสิร์ฟแล้ว)

2. **ระบบจัดการสำหรับแอดมิน (Admin POS Dashboard)**
   - รับออเดอร์และแจ้งเตือนเรียลไทม์เมื่อลูกค้าส่งรายการเข้ามา
   - ปุ่ม **"ชำระเงิน"** ดึงยอดเงินรวมไปคำนวณและสร้าง **PromptPay QR Code** อัตโนมัติ (สแกนจ่ายบนมือถือได้จริง)
   - ปุ่ม **"เสิร์ฟแล้ว"** สำหรับอัปเดตและเคลียร์หน้ากระดานรับออเดอร์
   - รายงานประวัติย้อนหลังและ **สรุปยอดขายรายวัน** แยกยอดของแต่ละวันพร้อมเมนูขายดีที่สุด
   - ระบบจัดการสินค้าคงคลัง (Stock Management) เพิ่ม/ลดสินค้า แก้ไขสต็อกสินค้า/ราคา และอัปโหลดไฟล์รูปภาพแนบในตัวได้
   - ระบบพิมพ์/ดาวน์โหลด QR Code ประจำแต่ละโต๊ะในร้านได้ทันทีผ่านหน้าแผงควบคุม

3. **การเก็บข้อมูลสองชั้น (Dual Database & Google Sheets)**
   - ทำงานแบบ Local JSON (Database สำรอง) ทันทีหลังเปิดโปรเจกต์
   - สามารถกรอก **Google Apps Script URL** เพื่อลิงก์ข้อมูลทั้งหมด (เมนู, สต็อก, ออเดอร์) ลงบน Google Sheets ของคุณ เพื่อใช้จัดการหรือแก้ไขผ่านสเปรดชีตได้แบบสดๆ

---

## วิธีติดตั้งและเริ่มใช้งาน

### 1. การเปิดใช้งานบนเครื่อง (Local Run)
1. เปิด Command Prompt หรือ PowerShell ขึ้นมา
2. เข้าไปยังโฟลเดอร์โปรเจกต์:
   ```bash
   cd C:\Users\piyaw\qrcode-menu
   ```
3. รันเซิร์ฟเวอร์ด้วยคำสั่ง:
   ```bash
   node server.js
   ```
4. เปิดหน้าต่างเว็บเบราว์เซอร์:
   - **สำหรับลูกค้า (หน้าสั่งอาหาร)**: เข้าไปที่ [http://localhost:3000](http://localhost:3000) (หรือระบุโต๊ะเช่น [http://localhost:3000/?table=1](http://localhost:3000/?table=1))
   - **สำหรับแอดมิน/ร้านค้า**: เข้าไปที่ [http://localhost:3000/admin.html](http://localhost:3000/admin.html)

*หมายเหตุ: ในการใช้งานจริงบนโทรศัพท์มือถือ ให้เชื่อมต่อมือถือเข้ากับ Wi-Fi เดียวกันกับเครื่องคอมพิวเตอร์ แล้วพิมพ์ IP Address ของคอมพิวเตอร์แทนคำว่า `localhost` เช่น `http://192.168.1.50:3000`*

## ขั้นตอนการเชื่อมต่อ Google Sheets

ระบบเชื่อมต่อฐานข้อมูลกับ Google Sheets ได้อย่างปลอดภัยผ่าน Google Apps Script โดยมีขั้นตอนดังนี้:

1. เปิด **Google Sheets** ของคุณและสร้างสเปรดชีตเปล่าขึ้นมา
2. บนแถบเมนูด้านบน เลือก **ส่วนขยาย (Extensions)** > **Apps Script**
3. คัดลอกโค้ดสคริปต์ในปุ่ม **"คัดลอกโค้ด Apps Script"** บนหน้าเว็บแอดมินแท็บตั้งค่า นำมาลบโค้ดเก่าใน Apps Script ออกทั้งหมดและวางลงไปแทนที่
4. กดบันทึก (รูปแผ่นดิสก์)
5. คลิกที่ปุ่ม **การทำให้ใช้งานได้ (Deploy)** > **การทำให้ใช้งานได้ใหม่ (New deployment)**
6. คลิกที่รูปเฟือง เลือก **เว็บแอป (Web app)**
7. ตั้งค่าการกำหนดค่าดังนี้:
   - **คำอธิบาย**: ใส่คำอธิบายใดๆ ก็ได้ เช่น "เชื่อมต่อร้านค้า QR"
   - **เรียกใช้ในฐานะ (Execute as)**: เลือก **ฉัน (Me / บัญชีอีเมลของคุณ)**
   - **ผู้มีสิทธิ์เข้าถึง (Who has access)**: เลือก **ใครก็ได้ (Anyone)** *(สำคัญมาก เพื่อให้เซิร์ฟเวอร์สามารถส่งข้อมูลเข้าสเปรดชีตได้)*
8. กด **Deploy (การทำให้ใช้งานได้)**
9. หากระบบขอสิทธิ์เข้าถึงบัญชี ให้กด **ให้สิทธิ์เข้าถึง (Authorize Access)** เลือกบัญชีอีเมลของคุณ กด **ขั้นสูง (Advanced)** และกด **ไปที่... (Go to... (unsafe))** แล้วกด **อนุญาต (Allow)**
10. ระบบจะแสดง **URL ของเว็บแอป (Web app URL)** ให้คลิกคัดลอก URL นั้น
11. นำ URL ไปวางลงในช่อง **"Google Apps Script Web App URL"** ในแท็บ **ตั้งค่าระบบ** ในหน้าแอดมิน แล้วกด **บันทึกการตั้งค่า**

หลังจากกดบันทึกสำเร็จ แถบมุมบนขวาจะเปลี่ยนเป็น **"เชื่อมต่อ Google Sheet แล้ว"** และระบบจะสร้างแท็บ **Menu** และ **Orders** บน Google Sheets ของคุณโดยอัตโนมัติ!
