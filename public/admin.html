เชื่อมต่อ Google Sheets (Apps Script Web App)

เมื่อเชื่อมต่อแล้ว ระบบจะดึง/บันทึกข้อมูลกับ Google Sheets โดยอัตโนมัติ

เครื่องมือสร้าง QR Code สำหรับตั้งโต๊ะ

ขั้นตอนการเชื่อมโยง Google Sheet

  1. สร้าง Google Spreadsheet ใหม่
  2. ไปที่เมนู ส่วนขยาย (Extensions) > Apps Script
  3. คัดลอกโค้ดจากคู่มือ Apps Script (สามารถกดดูโค้ดได้ในคู่มือด้านล่าง) นำไปวางแทนที่โค้ดทั้งหมดในหน้าต่างสคริปต์
  4. คลิกที่ปุ่ม การทำให้ใช้งานได้ (Deploy) > การทำให้ใช้งานได้ใหม่ (New deployment)
  5. เลือกประเภทเป็น เว็บแอป (Web app)
  6. ตั้งค่าสิทธิ์เข้าถึง: "ใครก็ได้" (Anyone) และเลือกทำงานเป็น "ฉัน" (Me)
  7. กด Deploy คัดลอก URL เว็บแอปที่ได้ มากรอกลงในช่องตั้งค่าด้านบนแล้วกดบันทึก
